# Color Snap 

A vibrant, logic-based memory matching game built with **SwiftUI**. Test your memory by matching colors in as few flips as possible!

##  Features (In Development)
* **Core Mechanics**: Interactive 2D grid matching logic.
* **Difficulty Levels**: Choose between Easy, Medium, and Hard modes.
* **Leaderboard**: Persistent high-score tracking using `UserDefaults`.
* **Dynamic Feedback**: Integrated Haptic Feedback and Progress Tracking.
* **MVVM Architecture**: Clean, scalable folder structure for professional development.

## Tech Stack
* **Language**: Swift 5.10
* **Framework**: SwiftUI
* **Architecture**: MVVM (Model-View-ViewModel)
* **Version Control**: Git & GitHub
